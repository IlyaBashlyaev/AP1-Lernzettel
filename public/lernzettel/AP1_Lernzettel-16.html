Umrechnung
DDR-RAM (Double Data Rate) bertrgt pro Takt zwei Datenpakete,
weshalb die effektive Datenrate doppelt so hoch ist wie die
Taktrate.
Taktfrequenz Datenrate:
Datenrate (MB/s) = Taktfrequenz (MHz) 2 Busbreite (Byte)
Typische Busbreite fr DDR-RAM: 64 Bit = 8 Byte
Beispiel:
DDR4-3200 (3200 MHz)
3200 2 8 = 51200 MB/s (= 51,2 GB/s)
