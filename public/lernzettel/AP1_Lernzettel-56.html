Symbol
Beschreibung
Objekt
Ein Objekt kann ein Informationsobjekt, ein
Material, eine Ressource oder ein Produkt sein.
Kontrollfluss
Gerichteter, zusammenhängender Graph, dessen
Knoten Ereignisse, Funktionen und
Verknüpfungsoperatoren sind.
Informationsfluss
Gibt an, dass anlässlich einer Funktion
Informationen von einem Informationsobjekt gelesen
oder auf ein Informationsobjekt geschrieben werden.
ODER
Material-/Ressourcenfluss
Gibt an, dass in einer Funktion Material bzw.
Ressourcen verbraucht werden.
Zuordnung
Gibt an, welche Organisationseinheit oder welche
Objekte einer Funktion zugeordnet sind.
Konnektor "UND"
Eine Funktion wird ausgeführt, wenn mehrere
Ereignisse eingetreten sind.
ODER
Nach einer Funktion treten mehrere Ereignisse ein.
ODER
Ein Ereignis tritt ein, nachdem alle direkt
vorangestellten Funktionen ausgeführt wurden.
