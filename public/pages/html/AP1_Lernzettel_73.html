8.3 Prozeduren und Funktionen
Einfache Funktionen
Einfache Funktionen haben keine bergabeparameter und keinen
Rckgabewert. Sie fhren nur Code aus.
Funktionen mit Rckgabewert
Funktionen mit Rckgabewert haben keine bergabeparameter, geben
aber einen Wert zurck. Beispiel: Getter-Methoden oder
Statusabfragen.
Funktionen mit bergabeparameter
Funktionen mit bergabeparameter akzeptieren einen oder mehrere
Parameter, geben aber keine Werte zurck. Parameter knnen

Funktionen mit Rckgabewert und
bergabeparameter
Diese Funktionen kombinieren beides: Sie akzeptieren Parameter
und geben einen Wert zurck. Dies ist die flexibelste Form von
Funktionen.
8.4 Objektorientierung
Objekt / Instanz
Eine Instanz ist ein konkretes Exemplar einer Klasse. Beispiel:
"meinAuto" ist eine Instanz der Klasse "Auto" mit individuellen
Attributwerten wie Farbe="rot".
Fugesteuerte Schleifen: Bedingung wird am Ende geprft
(do-while)
Call-By-Value: bergabe einer Wertkopie (Original bleibt
unverndert)
Call-By-Reference: bergabe einer Referenz/Adresse (Original
kann verndert werden)
