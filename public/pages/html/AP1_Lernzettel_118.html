11.19 Datensicherungsverfahren
Datensicherung (Backup) ist eine zentrale Manahme zur
Wiederherstellung von Daten im Falle von Hardware-Ausfllen,
Cyberangriffen oder menschlichen Fehlern.
Backup-Arten
Voll-Backup (Full Backup)
Inkrementelles Backup
Differenzielles Backup
Zugriffsrechte auf sensible Dateien und Verzeichnisse
einschrnken
Security Baselines (z. B. CIS Benchmarks, BSI IT-Grundschutz)
verwenden
Hrtungsskripte und Tools nutzen (z. B. Microsoft Security
Compliance Toolkit, Lynis fr Linux)
Schutz vor Cyberangriffen, Malware und Exploits
Minimierung von Sicherheitslcken durch reduzierte
Angriffsflche
Erfllung von Compliance-Anforderungen (z. B. DSGVO, ISO
27001)
Es wird eine komplette Kopie aller Daten erstellt.
Vorteil: Schnelle Wiederherstellung.
Nachteil: Hoher Speicherbedarf und lange Sicherungsdauer.
Es werden nur die seit dem letzten Backup genderten Daten
gesichert.
Vorteil: Schnelle Sicherung, wenig Speicherverbrauch.
Nachteil: Wiederherstellung dauert lnger, da mehrere Backup-
Stnde bentigt werden.
Sichert alle seit dem letzten Voll-Backup genderten Dateien.
